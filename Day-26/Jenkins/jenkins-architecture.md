# Jenkins Architecture

## Components:

### 1. Master Node
- Controls Jenkins
- Schedules jobs
- Manages pipelines

### 2. Agent Node
- Executes tasks
- Can run on different machines

### 3. Jobs
- Tasks or builds executed by Jenkins

### 4. Plugins
- Extend Jenkins functionality

---

## Workflow:
1. Developer pushes code
2. Jenkins triggers build
3. Code is tested
4. Application is deployed

---

## Benefits:
- Automation
- Continuous Integration
- Faster delivery
