# 🤖 Day 34 - Automation Script for Deployment

---

## 📌 Objective

* Automate deployment process
* Reduce manual work
* Implement rollback mechanism

---

## ⚙️ Scripts Included

### deploy.sh

* Builds Docker image
* Stops old container
* Runs new container

### rollback.sh

* Restores previous version

---

## ▶️ Run Deployment

```bash
bash deploy.sh
```

---

## 🔁 Rollback

```bash
bash rollback.sh
```
